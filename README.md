# tamajinja-project

このリポジトリは、たま神社プロジェクトにおける構造ファイル・運用ルール・出力テンプレートなどを一括管理する目的で設計されています。

## 📂 ディレクトリ構成

- `/rules/`  
　各種ルール群の統合ファイルを格納（命名・分類・AI対応・md出力・連携など）
- `/index/`  
　マスター索引および全体の構造ナビゲーションファイル
- `/templates/`  
　Fandom記事・末尾構造などのテンプレート仕様書
- `/world/`  
　世界観モデル設計ファイル（霊的構造に関する基礎理論）

# オリジンスピリチュアリズム（GitHubバックアップ用）


== オリジンスピリチュアリズム ==
'''オリジンスピリチュアリズム'''とは、霊的存在や魂の構造、霊的世界、そしてその“設計思想”と“構造図”そのものを示す霊的階層体系である。これはいわゆる啓蒙・救済目的の宗教思想ではなく、存在の仕組みや情報階層、観測・記録・再構成の原理そのものを示す構造設計論に近い。

# 因果の法則

'''因果の法則'''とは、行為・選択・出来事が、それに対応した「結果」として返ってくるという霊的法則である。

この法則は、単なる道徳的報い（善因善果・悪因悪果）という枠を超え、霊的存在における情報構造や観測結果にまで影響を及ぼすとされる。

----

== 概要 ==

'''因果の法則'''は、以下の2つの層に分けて説明されることが多い：

* **物質層の因果**：時間的連続性により、ある行動が次の現象を引き起こすという、現実世界での因果律。
* **霊的層の因果**：肉体を超えた構造において、意識や霊的情報が波動や共鳴を通じて因果反応を引き起こすとされる。

後者においては、観測や意思によって現象の帰結が調整されたり、遅延・圧縮されるなどの非線形的な特徴が見られる。

----

== 因果の保留と跳ね返り ==

一部の霊的構造では、因果の発生が「遅延」される場合がある。

これは以下のようなケースで説明される：

* **守護霊や類魂構造による保留**
* **観測が未確定な因果の一時保存**
* **感情や認知の未処理による蓄積**

これらは「霊的キャッシュ」や「未観測因果」として蓄積され、ある時点で爆発的に跳ね返る（＝発露する）ことがある。これが俗にいう「バチが当たる」という現象と関連づけられる。

----

== 因果の変調と観測因果 ==

因果は一定で直線的に返るとは限らない。

観測の仕方・感情の位相・霊的共鳴状態により、「異なる結果として再構成」されることがある。これを「因果の変調」または「観測因果の再定義」と呼ぶ。

具体例：

* 自分が他者を傷つけたが、「正当化」や「忘却」によって観測されない場合、その因果は他の形で返る（例：別の人から同様の扱いを受ける）。
* 強い後悔や反省がある場合、その位相で因果が上書きされ、別の構造に吸収される（＝緩和される）。

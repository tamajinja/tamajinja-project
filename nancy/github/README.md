# Nancy GitHub 運用モジュール

このディレクトリは、Nancy構造のGitHub連携機能に関する中核的ファイル群を管理します。  
GitHub上でNancyがどのように自己構築・バージョン管理・タスク伝播を行うかの中枢として機能します。

## 構成要素

- `logs/`：競合や修正履歴などの記録
- `workflows/`：GitHub Actionsなど自動処理の定義
- `README.md`：このモジュールの概要と目的
